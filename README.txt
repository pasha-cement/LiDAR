# Приложение для работы с LiDAR-датчиком

Десктопное приложение на Python для подключения к лазерному датчику измерения расстояния (LiDAR) JRT M703A и сбора данных с него. Приложение предоставляет удобный интерфейс для проведения измерений, визуализации данных в реальном времени и статистического анализа.

## Возможности

- Подключение к датчику LiDAR через последовательный порт (поддерживает JRT M703A)
- Выполнение единичных или непрерывных измерений расстояния
- Визуализация данных о расстоянии и качестве сигнала в реальном времени
- Статистический анализ измерений
- Сохранение и загрузка данных измерений в/из CSV-файлов

## Требования

- Python 3.7+
- PyQt5 (интерфейс)
- PySerial (подключение к датчику)
- Matplotlib (визуализация)
- NumPy (анализ данных)
- Pandas (обработка данных)

## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/pasha-cement/LiDAR.git
cd LiDAR
```

2. Установите необходимые зависимости:
```
pip install -r requirements.txt
```

## Использование

Запустите главное приложение:
```
python main.py
```

### Вкладка "Подключение"

1. Выберите последовательный порт, к которому подключен датчик LiDAR
2. Нажмите "Подключиться" для установления соединения с датчиком
3. Просмотрите информацию о датчике (версия, температура, напряжение)

### Вкладка "Измерение"

1. Выберите режим измерения (Единичное или Непрерывное)
2. Для непрерывного режима выберите скорость измерения (Авто, Быстро, Медленно)
3. Нажмите "Измерить" для начала измерений
4. Просматривайте текущие показания и историю измерений

### Вкладка "Визуализация"

1. Выберите тип графика (Расстояние, Качество сигнала, Скользящее среднее)
2. Настройте параметры визуализации (длина истории, автомасштабирование, сетка, маркеры)
3. Используйте панель инструментов matplotlib для навигации, масштабирования и сохранения графиков

### Вкладка "Статистика"

1. Просматривайте основную и расширенную статистику измерений
2. Нажмите "Обновить статистику" для обновления данных

## Технические характеристики датчика JRT M703A

### Основные параметры

| Параметр                     | Значение                           |
| ---------------------------- | ---------------------------------- |
| Бренд                        | JRT                                |
| Модель                       | M703A                              |
| Сертификаты                  | FDA/ISO9001/CE/FCC/ROHS            |
| Точность                     | ±1 мм (0.04 дюйма)                 |
| Диапазон измерения          | 0.03-40м                           |
| Время измерения              | 0.125~4 секунды                    |
| Класс лазера                 | Класс II                           |
| Тип лазера                   | 635нм, <1мВт                       |
| Размеры                      | 45x25x12мм (±1 мм)                |
| Вес                          | Около 10г                          |
| Напряжение                   | DC2.0~3.3B                         |
| Электрический уровень        | TTL/CMOS                           |
| Частота                      | 3Гц~8Гц                            |
| Рабочая температура          | 0~40°C (32~104°F)                  |
| Температура хранения         | -25~60°C (-13~140°F)               |

### Коммуникационный интерфейс

- Тип интерфейса: Последовательная связь (TTL)
- Скорость передачи: 19200 бод (19.2K)
- Бит чётности: Нет (без чётности)
- Биты данных: 8 бит

### Команды управления

| Команда | ASCII код (HEX) | Функция |
| ------- | --------------- | ------- |
| O       | 4F              | Включение лазера. Модуль возвращает ",ОК!" |
| C       | 43              | Выключение лазера. Модуль возвращает ",ОК!" |
| S       | 53              | Чтение состояния модуля. Возвращает "18.0'C, 3.0V" (температура и напряжение) |
| D       | 44              | Запуск автоматического измерения. Возвращает расстояние и качество сигнала (пример: "12.345m,0079") |
| M       | 4D              | Запуск медленного измерения с высокой точностью |
| F       | 46              | Запуск быстрого измерения с меньшей точностью |
| V       | 56              | Запрос информации о версии модуля |
| X       | 58              | Выключение модуля |

### Коды ошибок

| Код ошибки | Значение |
| ---------- | -------- |
| Er01!      | Напряжение питания слишком низкое (< 2.0V) |
| Er03!      | Температура модуля слишком низкая (< -20°C) |
| Er04!      | Температура модуля слишком высокая (> +40°C) |
| Er05!      | Цель вне диапазона измерения |
| Er06!      | Недействительный результат измерения |
| Er07!      | Фоновый свет слишком сильный |
| Er08!      | Лазерный сигнал слишком слабый |
| Er09!      | Лазерный сигнал слишком сильный |
| Er15!      | Лазерный сигнал нестабилен |
